# 💰 Expense Tracker

Track your daily expenses with this simple and responsive Expense Tracker built using **HTML**, **Tailwind CSS**, and **JavaScript**. This app allows you to log, display, and delete expenses, with persistent storage using `localStorage`.

### 🔗 Live Demo  
👉 [Open the App](https://expense-tracker-huzaifas-projects-827a906f.vercel.app/)

---

## 🚀 Features

- Add and list expenses with name and amount
- See total amount spent
- Delete individual expenses
- Responsive design (Mobile & Desktop)
- Data persistence using `localStorage`

---

## 🛠️ Built With

- **HTML5**
- **Tailwind CSS** for styling
- **Vanilla JavaScript** for functionality
- **localStorage** for data saving

---

## 🧠 How It Works

1. **Form Submission**  
   - Users enter expense name and amount
   - Data is validated and stored in localStorage
2. **Expense Display**  
   - Expenses are dynamically rendered into a list
3. **Deletion**  
   - Expenses can be removed individually with a click
4. **Total Calculation**  
   - Automatically updates total amount spent
